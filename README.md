# Competition & Raffle Platform

## Project Overview
A comprehensive online Raffle and Competition platform designed to provide a premium user experience. The platform allows users to browse competitions, purchase tickets via skill-based entry, and track their wins. 

## Features

### User Module
- **Secure Authentication**: Email/password and social login.
- **Wallet System**: Deposit funds, view transaction history, and earn cashback.
- **Ticket Management**: accurate tracking of active tickets and entry history.
- **Instant Wins**: Immediate notification system for instant prize allocation.

### Competition Engine
- **Dynamic Display**: Hero sliders, responsive grid views, and real-time status updates (Live, Ending Soon, Sold Out).
- **Skill-Based Entry**: Math/Trivia challenge required for compliance.
- **Live Stats**: Real-time tickets remaining and check functionality.
- **Cart System**: Discount codes and wallet balance integration.

### Admin Dashboard
- **Competition Management**: Full CRUD operations for raffles, including ticket supply and pricing.
- **Instant Win Setup**: Tools to generate and assign winning numbers.
- **Order & User Management**: User oversight and manual wallet adjustments.
- **Draw Management**: Export entry lists and declare winners.

## Tech Stack
- Frontend: Next.js / React (Planned)
- Styling: TailwindCSS / Custom CSS
- Backend: Node.js (Planned)
- Database: Postgres/SQL (Planned)

## Setup
1. Clone the repository.
2. Install dependencies: `npm install`
3. Run development server: `npm run dev`
